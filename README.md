# Web-Scraping

an attempt to web scrape linkeldn jobs

# Install

# server

make an .env file like that in .env.example

install the dependencies and start the server

```bash
npm install
npm run dev
```

# client

install the dependencies and start the server

```bash
npm install
npm start
```
